# DriveEase Mobile (Expo React Native)

This is the mobile version of DriveEase, built with Expo and React Native.

## Features
- Home screen with hero, stats, how it works, services, pricing, reviews, CTA, and about sections
- Ready for navigation and further feature expansion

## Getting Started
1. Install dependencies: `npm install`
2. Start the app: `expo start`

## Folder Structure
- `screens/` - App screens (Home, etc.)
- `components/` - Reusable UI components
- `navigation/` - Navigation setup
- `assets/` - Images and other assets
